# Helpers

Utility classes for WPF common tasks.

## Utilities

- `ObservableCollectionExtensions` — Extension methods for ObservableCollection
- `MsgBox` — Simplified message box interface
- `HebrewNumbering` — Hebrew numeral conversion and formatting
- `HebrewDateHelper` — Hebrew calendar date utilities
- `FontsHelper` — Font discovery and management
- `EventArgs<T>` — Generic event args for strongly-typed events
- `DependencyHelper` — Dependency injection and resolution helpers
- `ConfigurationManagerWrapper` — App.config settings wrapper
