# NakdanDemo

Standalone WPF demo application for testing Hebrew diacritical mark functionality.

## Purpose

Tests the Nakdan library independently of the Word VSTO context.

## Usage

1. Open `Nakdan.sln` in Visual Studio
2. Set `NakdanDemo` as the startup project
3. Press F5 to run

## Features

- Add diacritical marks (Nakdot) to Hebrew text
- Uses the Dicta API for processing
- No Word/Office required

Use this when developing diacritical mark features without needing Word open.
