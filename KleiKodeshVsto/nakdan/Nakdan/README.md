# Nakdan

WPF library for Hebrew diacritical mark (Nakdot) management using the Dicta API.

## Components

- `Core/` — Core diacritical mark processing engine
- `Helpers/` — Utility functions
- `UI/` — WPF user interface controls
- `WdStyles/` — Word document style management

## Features

- Adds diacritical marks to Hebrew text via Dicta API
- OOXML document parsing and manipulation
- Integration with Word through NakdanApi wrapper
