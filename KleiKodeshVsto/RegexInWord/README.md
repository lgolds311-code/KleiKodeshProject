# RegexInWord

Regex find & replace functionality for Word.

## Projects

### RegexFindLib
WPF class library providing:
- `RegexFindViewModel` — MVVM ViewModel for regex UI
- `RegexFindView` — WPF UserControl for the regex find/replace interface
- Full-featured regex find & replace with Word document integration

### RegexFindDemo
Standalone WPF demo application for testing regex functionality independently.

## Integration

The library is displayed as a task pane when the user clicks the "חיפוש רגקס" (Regex Find) ribbon button.

**Features:**
- Full regular expression support
- Replace with capture groups
- Case sensitivity options
- Whole words matching
