# WebSitesLib

WPF class library for the website browser component.

## Components

- `WebSitesView` — Main WPF UserControl with tabbed browser interface
- `BrowserTabControl` — Custom tab control for managing multiple browser tabs
- `MyWebView` — WebView2 wrapper component
- `WebAddressModel` — Data model for website entries
- `WebSitesWhitelist.json` — Configuration file listing available websites

## Integration

This library is packaged as a task pane displayed when the user clicks the "דרך האתרים" ribbon button in the VSTO add-in.
