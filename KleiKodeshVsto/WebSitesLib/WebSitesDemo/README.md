# WebSitesDemo

Standalone WPF demo application for testing the website browser component.

## Purpose

Tests the WebSitesLib library independently of the Word VSTO context.

## Usage

1. Open `WebSitesLib.sln` in Visual Studio
2. Set `WebSitesDemo` as the startup project
3. Press F5 to run

Use this when developing website browsing features without needing Word open.
