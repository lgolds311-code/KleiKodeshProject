# DocDesignLib

WPF class library for Torah document formatting.

## Components

- **Columns** — Multi-column layout for Torah text
- **Paragraphs** — Paragraph formatting controls
- **Spacing** — Line spacing and margin utilities
- **Fonts** — Hebrew font management

## Features

Used by the "עיצוב תורני" (Torah Formatting) ribbon button to format Word documents according to traditional Torah formatting conventions.
