# Helpers

Utility classes for the KleiKodesh VSTO add-in.

## Files

- `FormSettings.cs` — Settings form UI
- `JsonExtensions.cs` — JSON parsing utilities
- `MsgBox.cs` — Message box helper
- `OfficeThemeWatcher.cs` — Monitors Office application theme changes
- `SettingsManager.cs` — Manages user settings and preferences
- `TaskpaneManager.cs` — Manages task pane lifecycle and events
- `TaskPanePopOut.cs` — Handles task pane pop-out functionality
- `WdActionManager.cs` — Manages Word document actions
- `WordWindowHelper.cs` — Word window utilities
- `WpfTaskPane.cs` — WPF task pane wrapper for Word interop
