# כלי קודש למק (Office Add-in)

> **Mac companion add-in for Klei Kodesh.** A partial, web-based (Office.js)
> reimplementation of selected Klei Kodesh tools that runs on **Word for Mac**,
> where the main VSTO add-in (Windows-only) cannot run.

תוסף נלווה ל-Word **למק**, מבוסס Office.js. התוסף הראשי של כלי קודש בנוי
בטכנולוגיית VSTO שרצה רק על חלונות; תוסף זה מביא חלק מהיכולות גם למשתמשי מק.

## מה התוסף כולל

- **חיפוש והחלפה (רגקס)** — חיפוש והחלפה עם ביטויים רגולריים, מעבר ליכולות
  המובנות של וורד. עובד באופן עצמאי לחלוטין.
- **דיקטא וניקוד** — סימון טקסט במסמך וקבלת ניקוד אוטומטי משרת הניקוד של דיקטא
  ישירות בחזרה למסמך (ניסיוני — ראה הערות).
- **אתרים תורניים** — פתיחת דיקטא, נקדן פרו, נקדנית, ספריא, HebrewBooks ואוצר
  החכמה בחלון בתוך וורד.

## מבנה התיקייה

| קובץ | תיאור |
| --- | --- |
| `taskpane.html` | חלונית התוסף (עצמאי, כולל Office.js, RTL). |
| `manifest.xml` | מניפסט עם כפתור ברצועת הכלים, מצביע ל-GitHub Pages של הפרויקט. |
| `README.md` | מסמך זה. |

## אירוח (לבעלי הפרויקט)

מומלץ לארח את `taskpane.html` באתר הפרויקט (GitHub Pages), בתיקייה `mac/`, כך
שהכתובת תהיה `https://kleikodesh.github.io/mac/taskpane.html`. אז ניתן להפיץ
מניפסט אחד לכל המשתמשים. אם משנים את מיקום האירוח — יש לעדכן את כל כתובות ה-https
שב-`manifest.xml`. (אייקונים `icon-16/32/80.png` הם אופציונליים; בלעדיהם התוסף
עדיין יופיע, רק עם אייקון כללי.)

## התקנה (למשתמש מק)

1. הורד את `manifest.xml`.
2. ב-Finder לחץ `Cmd+Shift+G` והדבק:
   `~/Library/Containers/com.microsoft.Word/Data/Documents/wef`
   (אם התיקייה `wef` לא קיימת — צור אותה.)
3. העתק את `manifest.xml` לתוך התיקייה. **אל תדרוס מניפסטים אחרים** שכבר שם.
4. סגור את Word לגמרי (`Cmd+Q`) ופתח מחדש.
5. בלשונית **בית (Home)** תופיע קבוצה **"כלי קודש"** עם כפתור — לחץ עליו.

## הערות ומגבלות

- **הכפתור חובה:** המניפסט כולל `VersionOverrides` עם כפתור ברצועת הכלים. בלי
  זה התוסף **לא מופיע** ברשימת התוספים של Word למק.
- **ניקוד אוטומטי — ניסיוני:** משתמש בשרת הציבורי של דיקטא
  (`nakdan-*.loadbalancer.dicta.org.il`). אם דיקטא תשנה/תחסום את ה-API, הניקוד
  הישיר עלול להפסיק — ואז התוסף נופל אוטומטית לפתיחת נקדן פרו בחלון, עם הטקסט
  המסומן מועתק ללוח.
- **סינוני אינטרנט (נטפרי/אתרוג):** דומיין האירוח, ושרת הניקוד של דיקטא, עשויים
  לדרוש אישור בסינון לפני שהתוסף ייטען/יעבוד.
- **לא נכלל:** עיצוב תורני עמוק וקורא קיוויקס — אלו דורשים יכולות VSTO שאינן
  קיימות ב-Office.js במק.
