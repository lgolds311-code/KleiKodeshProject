# פתרון בעיית X אדום בכזית

## הבעיה
חלק מהמשתמשים רואים X אדום גדול במקום שהאפליקציה תיטען.

## פתרונות (לפי סדר חשיבות)

### 1. התקנת WebView2 Runtime (90% מהמקרים)
**הורד והתקן**: https://go.microsoft.com/fwlink/p/?LinkId=2124703  
בחר ב-"Evergreen Standalone Installer" והפעל מחדש את התוכנה.

### 2. אנטי-וירוס חוסם
כבה זמנית את האנטי-וירוס ונסה שוב. אם עובד - הוסף חריגה לתיקיית התוכנה.

### 3. קבצים חסרים
ודא שקיימת תיקייה `zayit-vue-app` עם קובץ `index.html` בתיקיית ההתקנה. אם לא - התקן מחדש.

### 4. מטמון פגום
סגור את התוכנה, מחק את התיקייה `ZayitWebView2SharedCache` שבתוך תיקיית ההתקנה (בתוך `zayit-vue-app`), והפעל מחדש.

---

**לדיווח תקלות צרף**: גרסת Windows, שם אנטי-וירוס, צילום מסך של השגיאה.
